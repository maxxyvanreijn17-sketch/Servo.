# NexDeploy

A production-grade cloud hosting platform website built with React, Vite, and Tailwind CSS.

## Tech Stack

- **React 19** — functional components with hooks
- **Vite 8** — fast build tooling (uses `@vitejs/plugin-react` with Babel for JSX)
- **Tailwind CSS 3** — utility-first styling with custom design tokens
- **Lucide React** — icon library

## Features

- Sticky glassmorphism navbar with mobile hamburger drawer
- Animated hero section with typewriter terminal component
- CSS marquee social proof bar
- Responsive features grid with scroll-triggered animations
- Pricing section with monthly/annual toggle
- Comparison table across all plan tiers
- FAQ accordion with smooth animations
- Fully responsive: mobile (375px), tablet (768px), desktop (1280px+)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
